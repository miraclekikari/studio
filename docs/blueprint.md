# **App Name**: LibreShare

## Core Features:

- User Authentication & Profiles: Secure user registration and login functionality, with individual user profiles to manage shared content and activity, leveraging Supabase for user data.
- Document Upload & Storage: Users can easily upload various document types, which are then securely stored and managed using Cloudinary, ready for sharing.
- Document Browsing & Search: A dynamic interface allowing users to browse through shared documents with essential search and filtering capabilities for easy discovery.
- In-App Document Viewer: Enables direct viewing of uploaded documents within the application, eliminating the need for external software for many file types.
- Document Download Functionality: Users can download any shared document directly from the platform for offline access or personal use.
- Personalized Recommendations: An AI tool that provides document recommendations to users based on their reading history, uploaded content, and community interactions, aiming to foster discovery and engagement.

## Style Guidelines:

- The primary color is a moderate, reassuring blue (#63A5DE), chosen to evoke trust and professionalism, setting a calm tone for a knowledge-sharing platform.
- The background color is a very light, cool grey-blue (#EDF4F9), providing a clean, accessible canvas that prioritizes readability and content focus, appropriate for a light color scheme.
- The accent color is a rich, deep indigo/violet (#3B1FA8), providing strong visual contrast for interactive elements and calls to action without being overly vibrant or distracting.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired feel. Body text font: 'Inter' (sans-serif) for exceptional readability and neutrality across all screen sizes, optimizing the document consumption experience.
- Utilize a consistent set of clean, line-based icons for clarity and intuitive navigation, especially for actions like upload, download, view, and user profile management.
- Implement a responsive, clean grid-based layout that prioritizes content visibility and ease of interaction across all devices, ensuring accessibility and a seamless experience for browsing and sharing documents.
- Employ subtle and swift UI animations for interactive elements, providing clear feedback on user actions (e.g., successful upload, button taps) without impeding performance or accessibility.
